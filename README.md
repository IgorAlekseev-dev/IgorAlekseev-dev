## 🚀 Ключевые проекты

В моем профиле представлены архитектурные демо-версии и pet-проекты. Часть коммерческого кода находится под NDA.

### 🏢[ГлавКрепеж — B2B E-commerce Platform](https://github.com/IgorAlekseev-dev/glavkrepezh)
Высокопроизводительная оптово-розничная платформа (5000+ SKU), построенная по архитектуре Modern Monolith.
* **Стек:** `FastAPI`, `PostgreSQL`, `SQLAlchemy (Recursive CTE)`, `HTMX`, `Docker`.
* **Достижения:** Ускорил обновление каталога с 16 часов до 5 секунд (кастомный парсер на Pandas). Внедрил отказоустойчивый поиск по опечаткам на базе `PostgreSQL FTS + pg_trgm`. SSR + HTMX позволили достичь LCP < 1 секунды.

### 🤖[Анонимный Telegram-бот](https://github.com/IgorAlekseev-dev/AskMessagesBot)
Микросервис для анонимного обмена сообщениями и маршрутизации транзакционных данных.
* **Стек:** `Python`, `Aiogram 3`, `PostgreSQL`, `Linux VPS`.
* **Достижения:** Спроектировал MVP, обеспечил надежную работу базы данных и маршрутизацию диалогов. Привлек 130+ активных пользователей, настроил production-ready развертывание на сервере (systemd/Docker).
